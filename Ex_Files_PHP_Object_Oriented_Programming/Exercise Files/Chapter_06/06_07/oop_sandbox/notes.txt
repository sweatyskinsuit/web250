Autoload Undefined Classes

- __autoload()
- Defined as a non-class function
- Called when PHP encounters an unknown class
- Provides an opportunity to locate the class definition

// PHP v4-5
function __autoload($class_name) {
    echo "Definition for {$class_name} is missing!";
}

$dog = new Pet();

solution:

// PHP v7
function my_autoload($class_name) {
    echo "Definition for {$class_name} is missing!";
}

spl_autoload_register('my_autoload');

$dog = new Pet();

You wouldn't want to echo back a simple string though, you'd want to actually look for the file:

function my_autoload($class) {
    if(preg_match('/\A\w+\Z/', $class)) {
        include 'classes/' . $class . ' .class.php';
    }
}

spl_autoload_register('my_autoload');

$dog = new Pet;

- Use autoload as a safety net
- Use autoload to conserve memory

Challenge: Inheritance

- Choose your own category
- Pick a topic that interests you 
- Look for subcategories with similarities, shared attributes
- Map inheritance on paper, list properties and methods

- Create new file challenge-02.php
- Define PHP classes
- Inherit, extend, and override properties and methods
- Create instances to test

Your Challenge Must
- Have one parent and at least 2 subclasses
- Classes must contain at least one class variable and one class methods
- Demonstrate that the subclass inherits from the parent class

Inheritance: When a new class takes on the properties and methods of an existing class.

Class Inheritance
- Organize code
- Prevent repetition
- Simplify maintenance
- Prevent inconsistencies and bugs

class ProductBrochure {
    var $file_name;

    function download_path() {
        return "/downloads/" . $this->file_name;
    }
}

class WarrantyDocument {
    var $file_name;

    function download_path() {
        return "/downloads/" . $this->file_name;
    }
}

class CleaningInstructions {
    var $filename;
}

- Behaviors (properties and methods) of a class are shared with subclasses
- Add new behaviors to parent and all subclasses gain them
- Subclasses can override parent behaviors
- Subclasses can extend parent behaviors
